# Merca Fácil

Aplicación web instalable para gestionar la lista semanal de Mercamadrid de Horno, Parador y Patio.

## Funciones

- Lista inicial importada desde `lista merca.xlsx`.
- Cantidades independientes por restaurante.
- Botón para marcar cada producto como pedido.
- Catálogo, búsqueda, altas y bajas.
- Asistente por texto y voz con instrucciones en español.
- Funciona sin conexión después de la primera apertura.
- Copias de seguridad y opción para compartir la lista.

## Instalación en Android

Abre la dirección publicada con Chrome, pulsa **Instalar aplicación** (o el menú de Chrome y **Añadir a pantalla de inicio**) y acepta. La voz necesita permiso de micrófono y una dirección segura `https://`.

Los datos se guardan solamente en el dispositivo. Conviene usar **Guardar copia** de vez en cuando.
